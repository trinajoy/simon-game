Press Any Key to Start

Object of the Game:

Correctly repeat a longer and longer sequence of signals.

  1. Press any key to start the game
  2. The game starts by flashing a square
  3. Repeat the pattern by clicking the square
  4. The game will duplicate the first pattern and add one
  5. Repeat the pattern by pressing the same color sqaures in order
  6. If you repeat the pattern incorrectly the game is over
